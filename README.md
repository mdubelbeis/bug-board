# Bug-Board

Bug-Board is a full-stack bug tracking application built with React, Express, and MongoDB.

This project is currently in early development.

## Project Goal

Build a portfolio-ready full-stack application for tracking software bugs across projects.

## Planned Features

- User authentication
- Project management
- Bug/issue tracking
- Bug status updates
- Priority and severity labels
- Comments on bugs
- Dashboard summary
- Search and filtering

## Tech Stack

### Frontend

- React
- Vite
- TypeScript

### Backend

- Node.js
- Express
- MongoDB
- Mongoose

### Tools

- pnpm
- Git/GitHub
- ESLint
- MongoDB Atlas
